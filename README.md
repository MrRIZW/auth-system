A full-stack authentication system built using the MERN stack.  
This project allows users to register, login, and access protected routes using JWT authentication.

---

Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Role Based Access Control (Admin/User)
- Secure Password Hashing
- REST API Integration
- Responsive UI

---

Tech Stack

Frontend
- React
- CSS
- Axios

Backend
- Node.js
- Express.js

Database
- MongoDB
- Mongoose

Authentication
- JSON Web Token (JWT)
- bcrypt
